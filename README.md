# NeuroVis — 神经网络结构可视化实验室

一个纯前端交互式网页，让用户通过拖拽组件搭建 CNN/ResNet 等经典网络结构，实时可视化数据流、特征图热力图、以及训练过程中的 loss/acc 曲线。

## 功能特性

### 🎨 网络构建
- **拖拽式组件**：从左侧面板拖拽神经网络组件到画布
- **丰富的组件库**：输入层、卷积层、池化层、激活函数、全连接层、归一化层等
- **可视化连接**：双击节点创建连接，直观展示数据流向
- **节点编辑**：选中节点后可编辑参数（滤波器数量、卷积核大小、步长等）

### 📊 可视化
- **数据流动画**：实时展示数据在网络中的流动过程
- **特征图热力图**：可视化每层输出的特征图
- **张量维度显示**：清晰展示每一层的输入输出张量形状变化

### 🚀 训练模拟
- **超参数配置**：学习率、批次大小、训练轮数、优化器选择
- **实时训练监控**：损失和准确率实时更新
- **训练曲线图表**：使用 Chart.js 绘制 Loss 和 Accuracy 曲线

### 📦 预置模板
- **LeNet-5**：经典手写数字识别网络
- **AlexNet**：ImageNet 冠军网络
- **VGG16**：深层卷积网络代表
- **ResNet18/50**：残差网络连接
- **EfficientNetV2**：高效网络架构

## 技术栈

- **HTML5**：页面结构
- **CSS3**：现代化深色主题 UI
- **JavaScript (ES6+)**：核心交互逻辑
- **SVG**：网络结构可视化
- **Canvas API**：数据流动画和热力图渲染
- **Chart.js**：训练曲线图表

## 使用方法

1. **打开应用**
   - 直接用浏览器打开 `index.html` 文件
   - 或使用本地服务器：`python -m http.server 8000`

2. **构建网络**
   - 从左侧组件面板拖拽组件到中间画布
   - 双击节点创建连接
   - 点击节点编辑属性

3. **加载模板**
   - 点击顶部"加载模板"按钮
   - 选择预设的经典网络架构

4. **配置训练**
   - 在底部训练面板设置超参数
   - 点击"开始训练"按钮

5. **可视化**
   - 点击导航栏的"可视化"或"训练监控"标签
   - 查看数据流动画、热力图和训练曲线

## 操作说明

### 画布操作
- **拖拽节点**：点击并拖动节点到新位置
- **创建连接**：双击一个节点，然后点击另一个节点
- **删除连接**：点击连接线，确认删除
- **缩放画布**：使用鼠标滚轮或点击缩放按钮
- **平移画布**：在空白区域按住鼠标拖动

### 快捷键
- `Delete`：删除选中的节点
- `Ctrl+Z`：撤销
- `Ctrl+Y`：重做

## 教育价值

NeuroVis 非常适合：
- **AI 入门教学**：直观展示神经网络的工作原理
- **网络架构学习**：理解不同层的作用和数据流向
- **超参数调优**：观察不同参数对训练的影响
- **可视化理解**：通过热力图和张量维度理解特征提取过程
