## 📚 使用指南

### 快速开始

1. **打开应用**
   ```bash
   # 方法1：直接打开
   双击 index.html
   
   # 方法2：使用本地服务器
   python -m http.server 8000
   # 然后访问 http://localhost:8000
   ```

2. **构建网络**
   - 从左侧面板拖拽组件到画布
   - 双击节点创建连接
   - 点击节点编辑属性

3. **加载预置模板**
   - 点击顶部"加载模板"按钮
   - 选择想要的网络架构
   - 自动生成完整的网络结构

4. **开始训练**
   - 在底部面板配置超参数
   - 点击"开始训练"按钮
   - 观察实时训练曲线

5. **查看可视化**
   - 点击导航栏的"可视化"标签
   - 选择不同的可视化类型：
     - **数据流动画**：查看数据在网络中的流动
     - **特征图热力图**：交互式探索特征图
     - **张量维度**：查看每一层的输出形状
     - **3D结构**：3D 视角查看网络结构

## 🎨 功能详解

### 1. 网络构建模块

#### 支持的层类型
- **输入层**：定义输入图像的形状
- **卷积层**：Conv2D、Conv3D、深度可分离卷积
- **池化层**：MaxPool、AvgPool
- **激活函数**：ReLU、Sigmoid、Softmax
- **全连接层**：FC、Dropout
- **归一化**：BatchNorm
- **输出层**：分类输出

#### 操作说明
- **添加节点**：从左侧拖拽到画布
- **移动节点**：拖拽节点到新位置
- **连接节点**：双击第一个节点，再点击第二个节点
- **删除连接**：点击连接线，确认删除
- **编辑属性**：点击节点，在右侧面板修改参数
- **删除节点**：选中节点，点击"删除节点"按钮

#### 画布操作
- **缩放**：鼠标滚轮或点击缩放按钮
- **平移**：在空白区域按住鼠标拖动
- **重置视图**：点击"重置视图"按钮

### 2. 预置模板

#### LeNet-5
- **应用场景**：手写数字识别（MNIST）
- **输入**：1×32×32 灰度图像
- **结构**：Input → Conv2D(6) → ReLU → MaxPool → Conv2D(16) → ReLU → MaxPool → FC(120) → ReLU → FC(84) → ReLU → Output(10)
- **特点**：经典的卷积神经网络结构

#### AlexNet
- **应用场景**：图像分类（ImageNet）
- **输入**：3×224×224 RGB 图像
- **结构**：5个卷积层 + 3个全连接层
- **特点**：使用 ReLU 激活、Dropout、数据增强

#### VGG16
- **应用场景**：图像分类
- **输入**：3×224×224 RGB 图像
- **结构**：13个卷积层 + 3个全连接层
- **特点**：全部使用 3×3 卷积核，网络深度增加

#### ResNet18
- **应用场景**：深层网络训练
- **输入**：3×224×224 RGB 图像
- **结构**：残差块（跳跃连接）
- **特点**：解决梯度消失问题，可以训练更深的网络

#### EfficientNetV2
- **应用场景**：高效图像分类
- **输入**：3×224×224 RGB 图像
- **结构**：MBConv 块（移动反转瓶颈卷积）
- **特点**：参数量少，准确率高

### 3. 训练功能

#### 超参数配置
- **学习率**：控制参数更新的步长（默认 0.001）
- **批次大小**：每次训练的样本数（默认 32）
- **训练轮数**：完整数据集的训练次数（默认 50）
- **优化器**：Adam（推荐）、SGD、RMSprop

#### 训练监控
- **实时指标**：损失（Loss）和准确率（Accuracy）
- **进度条**：显示当前训练进度
- **训练曲线**：使用 Chart.js 绘制 Loss 和 Accuracy 曲线
- **训练控制**：开始、暂停、停止

#### TensorFlow.js 训练
- 自动构建 TensorFlow.js 模型
- 使用实际的反向传播算法
- 支持 GPU 加速（如果可用）
- 显示真实的训练指标

### 4. 可视化功能

#### 数据流动画
- **功能**：展示数据在网络中的流动过程
- **特点**：
  - 动态箭头指示数据方向
  - 张量形状实时更新
  - 流动粒子效果
- **用途**：理解前向传播过程

#### 特征图热力图
- **功能**：可视化每层输出的特征图
- **特点**：
  - 颜色编码表示激活强度
  - 鼠标悬停显示具体数值
  - 交互式探索
- **用途**：理解卷积层学到了什么特征

#### 张量维度展示
- **功能**：显示每一层的输入输出张量形状
- **特点**：
  - 清晰的维度信息
  - 参数说明
  - 颜色编码
- **用途**：理解数据在网络中的形状变化

#### 3D 网络结构
- **功能**：3D 视角展示网络结构
- **特点**：
  - 球体代表节点
  - 曲线连接表示数据流
  - 可旋转、缩放
  - 自动旋转动画
- **用途**：直观理解网络的整体架构

## 🐛 已知问题

1. **TensorFlow.js 加载**
   - 如果 TensorFlow.js CDN 加载失败，会自动降级到模拟训练
   - 解决方法：检查网络连接，或使用本地 TensorFlow.js 文件

2. **Three.js 性能**
   - 节点数量过多时，3D 可视化可能会卡顿
   - 解决方法：减少节点数量，或升级硬件

3. **浏览器兼容性**
   - 推荐使用 Chrome 90+ 或 Firefox 88+
   - Safari 可能需要启用 WebGL 2.0
